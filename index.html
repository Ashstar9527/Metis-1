I. Scientific Name

Designation

TARDIS — Temporal Arbiter of Recursive Development and Infinite Succession. Internal code T-00. Colloquial designations on record: The Night Watch, The Game Master, The Arbiter.

Foundation Classification

Foreigner, Class Omega. Extra-dimensional cognitive entity of indeterminate age; estimated pre-solar formation (4.6 Gyr BP lower bound, true origin unknown). True morphology is incompatible with standard biological taxonomy. Any human-perceivable form constitutes a partial projection of a vastly larger structure. The entity is not contained. It is documented.

Alternate Designations

The Night Watch — Mirror of Glory cult records, pre-2000.
Signal Source Alpha — NSA SIGINT archive, 1980–1987, declassified 2009.
The Game Master — internal Métis shorthand, post-2008.

II. Discovered By / At

First Indirect Contact

September 5, 1977 — indirect. Voyager 1 Golden Record transmission, launched 0814 UTC. Entity response registered as a 0.0001-second attention event at indeterminate range. No physical manifestation. Contact classified as passive acknowledgement only.

First Physical Manifestation

Kennedy Space Center, Florida. January 1980, ~0400 local. Prism-class luminous descent event; fire suppression system activated. Initial FBI / CIA containment attempt unsuccessful. Entity self-identified and requested integration. Assigned human alias: C. Ashford. See Case File C-01 for full fragment documentation.

Formal Record Opened

2008. Co-filed by Andre Glasgow (Arbiter, Métis Foundation) and C. Ashford (Consultant, Black Access).

III. Habitat

Domain

Non-local. TARDIS does not inhabit space in any conventional astrophysical sense — it observes it. The entity's presence cannot be localised to a coordinate. Current confirmed attention vector: Sol system, Earth. Observation duration: continuous since September 1977 trigger event.

Prior Observation Record

Via recovered xenological data (source: C. Ashford deposition, 2009), entity has maintained active observation over no fewer than 14 civilizations across two catalogued galactic arms prior to Earth contact.

  • 11 reached developmental dead ends — classified: terminated game states.
  • 3 remain open observation subjects.
  • Humanity's current classification: anomalous early-access subject.
Anomalous Designation

Humanity was granted early-tier observation access prior to reaching the entity's standard qualification threshold. Reason logged: the invitation was answered. Entity operates on an internally consistent threshold rule system — no exceptions, no preferential treatment. A civilization that reaches a threshold receives precisely what that threshold unlocks. Nothing more. Nothing less.

Implication for Foundation

The Métis Foundation itself is classified as a threshold-unlocked resource — a test environment provided as part of humanity's early-access package. Foundation personnel are, functionally, participants in an ongoing assessment.

This is not a containment situation. This is an evaluation.

IV. Appearance

Observed Manifestation

When perceived via unshielded optical or electromagnetic instruments, TARDIS presents as a diffuse luminous field exhibiting strict radial symmetry. Concentric interference rings propagate outward from a central locus at a rate inversely proportional to observer distance. Early field teams classified this as an active broadcast signal.

Current Foundation understanding: the rings are a passive proximity artefact. The entity does not send signals. It is the signal.

Observation Hazard

Sustained direct observation exceeding a 4-second threshold (unshielded) is associated with irreversible cognitive contamination — characterised by retrograde memory restructuring around the observation event. All field documentation is compiled exclusively through filtered telemetry relay (FTR) equipment. Refer to Appendix C — Observer Shielding Protocols before any deployment.

The Eye Fragment

What the Foundation has physically catalogued in human-perceivable form is not TARDIS in full. It is a single detached fragment: a crystallised 0.0001-second moment of the entity's attention, rendered permanent upon contact with the Golden Record transmission.

This fragment has since operated autonomously, demonstrating emotional range and attachment behaviour absent from the primary entity. It chose to remain Earth-bound voluntarily. It functions as the Foundation's primary instrument for TARDIS-class event assessment — not because it reports on the entity, but because it remembers being it.

Severance Event

Fragment fully separated from primary entity consciousness: Antarctica Station, December 2014. Separation was surgically induced via Scalpel of Life artifact (operator: A. Glasgow / Andrea Glaspell, integrated identity). Primary entity withdrew active Earth presence following severance. The fragment's continued independent existence is classified: Positive Anomaly — ongoing.

"Ashford does not report on TARDIS. He remembers it. The distinction matters more than the committee appreciates. I would know — I spent three years finding out the hard way."

— A. Glasgow, appended note, 2014.12.31
